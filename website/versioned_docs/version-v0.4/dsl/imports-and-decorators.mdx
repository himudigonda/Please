# Imports and Decorators

## Imports

```text
@import backend/broskifile
@import frontend/broskifile
```

Guardrails:

- circular imports are rejected with a readable chain
- collisions across task/alias/variable names fail hard
- depth limit prevents runaway recursion

## Decorators

```text
@private
@confirm "Ship release artifacts? [y/N]"
```

- `@private` hides tasks from standard listings
- `@confirm` protects sensitive workflows
