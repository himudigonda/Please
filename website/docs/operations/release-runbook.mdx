# Release Runbook

1. Ensure local gates are green.
2. Merge to `main`.
3. Tag release (`vX.Y.Z`).
4. Verify release workflow and artifacts.
5. Validate install script and `broski --version`.
6. Publish notes and update portal links.

See repository runbook for exact command sequence and rollback actions.
